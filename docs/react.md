# REACT

## Criar a Aplicação React

```bash
npx create-react-app horadoqa
cd horadoqa
```

## Startar a aplicação

```bash
npm start
```

## Instalar o Axios

Você pode usar o axios para fazer requisições HTTP. Instale-o no seu projeto React:

```bash
npm install axios
```

Inicie o frontend: No diretório do seu frontend, execute:

```bash
npm start
```
